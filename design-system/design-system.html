Component Class Patterns

// Button variants
btn-primary   → bg-accent text-fg-inverse px-5 py-2 rounded-sm font-medium text-sm
btn-secondary → bg-transparent text-fg border border-border px-5 py-2 rounded-sm text-sm
btn-ghost     → bg-transparent text-fg-secondary px-2 py-2 rounded-sm
btn-icon      → w-9 h-9 rounded-full flex items-center justify-center

// Card
card          → bg-surface rounded-md border border-border overflow-hidden
card-body     → p-4
card-hover    → hover:shadow-md hover:border-border-strong transition-all

// Input
input         → w-full px-4 py-2 border border-border rounded-sm bg-surface text-fg
input-focus   → focus:border-accent focus:ring-3 focus:ring-accent/10

// Tab
tab           → px-4 py-3 text-sm font-medium text-fg-muted
tab-active    → text-fg after:absolute after:bottom-0 after:left-0 after:right-0 after:h-0.5 after:bg-accent
